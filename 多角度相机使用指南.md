# 多角度相机使用指南

## 快速开始

### 1. 创建节点
双击画布空白处 → 选择"多角度相机"

### 2. 基础操作

#### 🖱️ 拖拽旋转
- **左右拖动**：旋转人物（水平 0° - 360°）
- **上下拖动**：倾斜视角（垂直 -30° - 60°）
- **提示**：光标会变成抓手图标

#### 🔍 滚轮缩放
- **向上滚动**：相机拉近（特写）
- **向下滚动**：相机拉远（全景）
- **范围**：0 - 10

#### 🎚️ 滑块精确调整
- **水平滑块**：0° - 360°，步进 1°
- **垂直滑块**：-30° - 60°，步进 1°
- **距离滑块**：0 - 10，步进 0.1

### 3. 快速预设

#### 水平方向（8 个预设）
- ⬆️ **正面**（0°）：面向观众
- ↗️ **右前**（45°）：右前方 45°
- ➡️ **右侧**（90°）：完全侧面
- ↘️ **右后**（135°）：右后方 45°
- ⬇️ **背面**（180°）：背对观众
- ↙️ **左后**（225°）：左后方 45°
- ⬅️ **左侧**（270°）：完全侧面
- ↖️ **左前**（315°）：左前方 45°

#### 垂直方向（4 个预设）
- 📐 **低角度**（-30°）：从下往上看（仰拍）
- 👁️ **平视**（0°）：水平视角
- 📈 **仰角**（30°）：略微俯视
- 🔭 **高角度**（60°）：从上往下看（俯拍）

#### 距离（3 个预设）
- 🔍 **特写**（2）：近距离拍摄
- 📷 **中景**（5）：标准距离
- 🌄 **远景**（8）：远距离拍摄

### 4. 预设组合

#### 标准三视图
- 正面（0°, 0°, 5）
- 侧面（90°, 0°, 5）
- 背面（180°, 0°, 5）

#### 360° 环绕
- 8 个角度，每 45° 一个
- 平视，中景距离

#### 多角度特写
- 6 个角度
- 不同垂直角度
- 特写距离

## 3D 场景说明

### 场景元素

#### 👤 人物模型
- **头部**：圆形，肤色，有五官
- **身体**：蓝色衣服，矩形
- **手臂**：左右各一条，略微倾斜
- **腿部**：深色裤子，站立姿势

#### 📷 相机
- **位置**：根据参数动态移动
- **图标**：紫色相机图标
- **光晕**：紫色发光效果
- **标签**："Camera" 文字标签

#### 📐 取景框
- **尺寸**：80×60px（16:9 比例）
- **位置**：相机前方
- **角标**：四个角的 L 形标记
- **十字**：中心对焦线

#### 🌐 地面网格
- **尺寸**：500×500px
- **网格**：40×40px 方格
- **颜色**：紫色发光线条

#### 📊 坐标轴
- **X 轴**：红色，水平向右
- **Y 轴**：绿色，垂直向上
- **Z 轴**：蓝色，垂直向前

### 参数显示

#### 实时参数
- **H（水平）**：紫色，当前水平角度
- **V（垂直）**：青色，当前垂直角度
- **Z（距离）**：粉色，当前缩放值

#### 预设标签
- 显示当前匹配的预设名称
- 例如："正面 · 平视 · 中景"

## 使用场景

### 场景 1：人物肖像
```
目标：拍摄人物正面肖像
操作：
1. 选择"正面"预设（0°）
2. 选择"平视"预设（0°）
3. 选择"特写"预设（2）
4. 微调：拖拽稍微向右（5°）
5. 点击"生成多角度图片"
```

### 场景 2：产品展示
```
目标：360° 展示产品
操作：
1. 点击"360° 环绕"预设组合
2. 自动生成 8 个角度
3. 查看 9 宫格结果
4. 选择最佳角度
```

### 场景 3：动作捕捉
```
目标：捕捉动作的多个角度
操作：
1. 选择"多角度特写"预设组合
2. 生成 6 个不同角度
3. 在 9 宫格中选择
4. 用于后续视频生成
```

### 场景 4：建筑摄影
```
目标：拍摄建筑物
操作：
1. 选择"高角度"预设（60°）
2. 选择"远景"预设（8）
3. 拖拽调整水平角度
4. 找到最佳构图
```

## 常见问题

### Q1：拖拽不灵敏？
**A**：拖拽灵敏度已优化：
- 水平：0.5x（更平滑）
- 垂直：0.3x（更精确）
- 如需更快，可以用滑块

### Q2：滚轮缩放冲突？
**A**：已阻止事件冒泡：
- 在 3D 场景内滚动 = 相机缩放
- 在场景外滚动 = 画布缩放

### Q3：看不清人物细节？
**A**：调整相机距离：
- 滚轮向上 = 拉近
- 或选择"特写"预设
- 或拖动距离滑块到 2

### Q4：角度不够精确？
**A**：使用滑块：
- 水平：步进 1°
- 垂直：步进 1°
- 距离：步进 0.1

### Q5：预设不够用？
**A**：自定义角度：
- 拖拽到任意角度
- 或用滑块精确调整
- 参数会实时显示
