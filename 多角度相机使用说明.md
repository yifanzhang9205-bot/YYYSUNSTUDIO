# 多角度相机节点使用说明

## 功能介绍

多角度相机节点可以为同一对象生成多个不同角度的视图，非常适合用于：
- 角色设计的多角度展示
- 产品展示的 360° 视图
- 3D 建模参考图生成
- 分镜中保持角色一致性

## 使用方法

### 1. 创建节点

双击画布空白处，在弹出菜单中选择：
```
高级工具 → 多角度相机
```

### 2. 连接输入（可选）

可以从以下节点连接到多角度相机：
- **图片生成节点** - 为生成的图片创建多角度视图
- **图像编辑节点** - 为编辑后的图片创建多角度视图

如果不连接输入，多角度相机会根据提示词从零生成。

### 3. 调整相机参数

#### 快速预设
使用三个下拉菜单快速选择常用角度：

**水平方向（8个选项）：**
- 正面（0°）
- 右前（45°）
- 右侧（90°）
- 右后（135°）
- 背面（180°）
- 左后（225°）
- 左侧（270°）
- 左前（315°）

**垂直角度（4个选项）：**
- 低角度（-30°）- 从下往上看
- 平视（0°）- 水平视角
- 仰角（30°）- 略微向上
- 高角度（60°）- 从上往下看

**距离（3个选项）：**
- 特写 - 近距离拍摄
- 中景 - 中等距离
- 远景 - 远距离拍摄

#### 精确控制
使用滑块进行精确调整：
- **水平角度**：0° - 360°（任意角度）
- **垂直角度**：-30° 到 60°（精确到 1°）
- **距离**：0 - 10（精确到 0.1）

### 4. 添加自定义提示词（可选）

在"自定义提示词"文本框中，可以添加额外的描述：
- 光照效果：如"soft lighting", "dramatic shadows"
- 材质质感：如"metallic surface", "fabric texture"
- 风格要求：如"photorealistic", "anime style"
- 环境设置：如"outdoor", "studio background"

### 5. 生成多角度图片

点击"生成多角度图片"按钮，系统会：
1. 根据相机参数生成标准化的提示词
2. 结合你的自定义提示词
3. 生成 9 张不同角度的候选图片（9宫格）

### 6. 选择最佳结果

在 9 宫格中点击选择你最满意的角度图片。选中的图片会：
- 显示紫色边框和光晕
- 可以连接到下游节点使用

## 相机提示词格式

节点会自动生成符合 Qwen-Image-Edit-2511-Multiple-Angles-LoRA 格式的提示词：

```
azimuth: [方向], elevation: [角度], distance: [距离]
```

例如：
```
azimuth: front-right quarter view, elevation: eye-level shot, distance: medium shot
```

## 工作流示例

### 示例 1：角色多角度设计
```
创意工作室 → 角色参考 → 多角度相机 → 分镜生成
```
1. 在创意工作室生成角色概念
2. 在角色参考中选择最佳设计
3. 使用多角度相机生成 360° 视图
4. 在分镜生成中使用一致的角色

### 示例 2：产品展示
```
图片生成 → 多角度相机 → 视频生成
```
1. 生成产品图片
2. 创建多个角度的产品视图
3. 组合成旋转展示视频

### 示例 3：3D 建模参考
```
草图 → 图片生成 → 多角度相机 → 导出
```
1. 手绘草图
2. AI 生成精细图片
3. 生成多角度参考图
4. 导出用于 3D 建模软件

## 技巧和建议

### 1. 保持一致性
- 使用相同的自定义提示词确保所有角度风格一致
- 从已有图片连接输入可以保持对象特征

### 2. 选择合适的角度
- **角色设计**：建议使用正面、侧面、背面三个主要角度
- **产品展示**：建议使用 45° 间隔的 8 个角度
- **建筑物**：建议使用低角度或高角度突出气势

### 3. 调整距离
- **特写**：适合展示细节、表情、纹理
- **中景**：适合展示整体造型
- **远景**：适合展示环境关系

### 4. 优化提示词
好的自定义提示词示例：
```
professional photography, studio lighting, white background, 
high detail, 8k resolution, product shot
```

避免的提示词：
- 过于复杂的场景描述
- 与相机角度冲突的方向词（如"from above"）
- 多个对象（会导致角度混乱）

## 常见问题

### Q: 生成的图片角度不准确怎么办？
A: 尝试：
1. 使用快速预设而不是手动调整
2. 简化自定义提示词
3. 确保输入图片背景简洁

### Q: 9 张图片差异太大怎么办？
A: 这是 AI 生成的特性，建议：
1. 从已有图片连接输入（而不是从零生成）
2. 在自定义提示词中强调一致性
3. 多次生成，选择最佳批次

### Q: 可以生成更多或更少的图片吗？
A: 当前版本固定生成 9 张图片。未来版本会支持自定义数量（4/9/16 宫格）。

### Q: 如何导出所有角度的图片？
A: 目前需要：
1. 选择每个角度
2. 右键点击节点
3. 选择"下载"
4. 重复以上步骤

未来版本会支持批量导出。

## 快捷键

- **右键节点**：打开上下文菜单
