# 多角度相机 - 全屏展示升级完成

## 升级概述
将多角度相机节点从可收起的小尺寸升级为始终展开的全屏大尺寸显示，充分展示 3D 视角效果。

## 主要改动

### 1. 节点高度升级
**文件**: `App.tsx` - `getApproxNodeHeight` 函数

```typescript
if (node.type === NodeType.MULTI_ANGLE_CAMERA) {
    // 多角度相机：始终展开，大尺寸显示

}
```

**文件**: `components/Node.tsx` - `getNodeHeight` 函数



## 技术细节

### CSS 3D Transform
```css
perspective: 1200px;  /* 增强透视 */
transform-style: preserve-3d;
transform: rotateX(${-verticalAngle * 0.5}deg) rotateY(${horizontalAngle}deg);
transition: transform 0.7s ease-out;  /* 平滑过渡 */
```

### 立方体六面体
- 前面：translateZ(60px)
- 后面：translateZ(-60px) rotateY(180deg)
- 左面：rotateY(-90deg) translateZ(60px)
- 右面：rotateY(90deg) translateZ(60px)
- 上面：rotateX(90deg) translateZ(60px)
- 下面：rotateX(-90deg) translateZ(60px)

### 相机位置计算
```typescript
transform: `
  translate(-50%, -50%)
  rotateY(${-horizontalAngle}deg)
  rotateX(${verticalAngle}deg)
  translateZ(${180 + cameraZoom * 20}px)
  rotateY(${horizontalAngle}deg)
  rotateX(${-verticalAngle}deg)
`
```
